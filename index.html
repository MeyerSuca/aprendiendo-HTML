Aprendiendo HTML

Es un lenguaje de demarcado para esquematizar un sitio web, entiendase como el esqueleto del sitio web

Etiquetas "h1",..."h6"

Sirven para la creación de titulos, según el numero será el tamaño del titulo

Datos: solo debe de haber una etiqueta h1 en una pagina, el resto si, cuantos quieras.

Etiqueta "p"

Esta etiqueta sirve para la creacion de texto en el sitio web

Etiquetas Anchor "a"

Esta etiqueta sirve para la creación de enlaces, todo lo que este dentro de esta etiqueta será un enlace

Etiquetas: unsorted list"ul" y order list"ol"

Etiqueta que sirve para la creacion de listas, ya sea ordenadas o desordenadas

Etiqueta "li"

Esta etiqueta debe de estar dentro de las etiquetas "ol" o "ul", sirve para agregar elementos, cada uno es un elemento

Etiqueta "div"

Sirve como un contenedor para agrupar elementos y estructurar mejor nuestras paginas

Etiqueta "header"

Esta etiqueta sirve para crear un encabezado, dentro de esta etiqueta van el logo o nombre de la pagina, tambien la barra de navegacion

Etiqueta: "footer"

Esta etiqueta sirve para crear el pie de pagina, lugar donde van informacion general, derechos de autor, terminos y condiciones, etc

Etiqueta de imagen: "img"

Esta etiqueta sirve para insertar imagenes en nuestro sitio web

Etiqueta de tablas: "table"

Etiqueta para la creacion de tablas

Etiqueta: "tr"

etiqueta que crea columnas dentro de la tabla

Etiqueta: "th"

Etiqueta que crea cabecera de una tabla, para agregar titulos a las columnas

Etiqueta: "td"

Etiqueta que crea filas, para agregar contenido

Formulario: "form"

Etiqueta para la creacion de formularios

Etiqueta: "Input"

Sirve para la creacion de espacios para recavar datos del usuario

"Label"

Etiqueta que sirve para asociar a inputs y es donde van nombres del espacio al que se asocia

Aprendiendo CSS

Es un lenguaje de estilos que se encarga de la parte visual que ve el usuario, CSS es quien se encarga de dar una buena presentacion al usuario.

Enlazar elementos HTML con CSS

Para enlazar se recomienda usar clases o id, segun sea el contexto, siendo clases la manera mas usada en general, id es usado en caso se quiera aplicar a un solo elemento, tambien se pueden usar otros tipos de enlazado, pero clases y id sera más usado

Aprendiendo Javascript

Javascript es un lenguaje de programacion para la implementacion de dinamicidad a un sitio web

Enlazar un elemento HTML a javascript

para enlazar un elemento se usa un id de nombre unico que debera tener la etiqueta, tambien se puede enlazar por su nombre, clase, etc, pero recomiendan enlazarlo por un id en general
